# 地震學 (Seismology) 課堂重點整理

## 1. 地震學概論與應用
### 什麼是地震學？
* **定義**：研究地震以及地震波特性的科學。
* **研究內容**：地震成因、造成的效應、地震波在地球（或星體）內部的傳播方式。
* **目的**：
    * 理解地球內部構造、物理性質與組成。
    * 應用地震儀記錄並分析振動行為。
    * **振動來源**：不限於斷層錯動，還包括火山噴發、爆炸事件、暴風、大型山崩等。
* **應用領域**：
    * 地震災害評估、資源探勘、理解地球動力行為、地震預報。
    * **環境監測**：山崩/落石監測、地下水位監測、颱風/暴風軌跡、水體振盪幅度。

## 2. 地震發生的物理機制
### 彈性回跳理論 (Elastic Rebound Theory)
* 由 Harry Fielding Reid (1906) 提出。
* **過程**：
    1.  **原來位置**：岩層處於未受力狀態。
    2.  **累積應變**：板塊運動導致岩層變形，累積能量。
    3.  **滑移錯動 (地震)**：累積的應變克服了斷層面的摩擦力，瞬間錯動。
    4.  **應變釋放**：能量以「彈性波」形式向四面八方輻射，岩層回到無應變但位移後的位置。
* **黏滑運動行為 (Stick-Slip Behavior)**：
    * **地震機器 (Earthquake Machine)** 模型顯示：震間累積應力 (Stress Accumulation) $\rightarrow$ 地震時應力下降 (Stress Drop)。
    * 若已知應力累積速率和摩擦力，理論上有機會預測重複週期，但實際變因太多。

### 震源物理
* 涉及剪切應力 (Shear Stress)、剪切應變 (Shear Strain)、岩石強度與斷層強度。
* **為什麼地震難以預測？**
    * 雖然基本物理過程已知，但細節異常複雜。
    * **應力累積速率不固定**：受鄰近地震影響。
    * **斷層強度非定值**：流體遷移可能弱化地殼，癒合程度因時而異。
    * 每次地震的應力降程度也不一。

## 3. 地震波與振動方式
### 體波 (Body Waves)
* **P波 (縱波)**：跑得快，介質振動方向與傳播方向平行（類似排隊推人）。
* **S波 (橫波)**：跑得慢，介質振動方向與傳播方向垂直（類似波浪舞），通常搖晃較大，破壞力強。

### 表面波 (Surface Waves)
* 主要在近地表傳播，能量不易消散，振幅通常更大。
* **種類**：雷利波 (Rayleigh Wave)、洛夫波 (Love Wave)。

### 波速公式
* $V_p = \sqrt{\frac{\lambda + 2\mu}{\rho}}$
* $V_s = \sqrt{\frac{\mu}{\rho}}$
    * ($\rho$: 密度, $\mu$: 剪切模量)

## 4. 台灣地震觀測與定位
### 觀測歷史
* **儀器觀測前**：依賴文獻記載（歷史記載地震）。
* **1897年**：台北測候所裝設第一部地震儀（格雷-米爾恩型），開啟儀器觀測時代。
* **1935年後**：全台約有10個地震站（台北、台中、阿里山、台南、高雄等）。
* **1973年**：中研院建立 TTSN (台灣遙記式地震觀測網)。
* **1991年**：氣象署升級為 CWASN (即時地震觀測網)，合併 TTSN，測站增至 75 站以上。

### 震央定位原理
* **距離與時間的關係**：
    * P波與S波的到時差 ($T_s - T_p$) 與震央距離成「線性正比」關係。
    * **簡易公式**：$\Delta (\text{公里}) \approx 8 \times (T_s - T_p)$
    * 早期人工定位公式 (1916南投地震範例)：$\Delta = 6.86 \times (T_s - T_p) + 8.1$
* **RMT 系統**：即時地震矩張量監測，可分析震源機制。

### 地殼變形監測
* **GPS**：觀測地表變形量 ($\delta X$) 與應變量 ($\delta X / X$)，找出累積應變能的位置。
* **雷達衛星干涉 (InSAR)**：利用衛星 (如 Sentinel-1A) 監測大範圍的地表變形（如簡報中的彩色干涉圖）。

## 5. 地震預警系統 (EEW)
* **原理**：利用電波（光速）快於地震波（約 3~6 km/s）的特性。
* **運作**：偵測到跑得快的 P 波後，在破壞性強的 S 波到達前，預估地震位置、規模與各地震度並發出警報。
* **盲區 (Blind Zone)**：震央附近的區域因距離太近，系統運算與傳輸不及，無法提供預警。
* **現況**：引入機器學習與人工智慧模型優化。

## 6. 地震統計與災害風險
### 統計特性
* **古騰堡-芮克特法則 (Gutenberg-Richter Law)**：地震規模與數量呈現對數線性關係（規模越小，數量越多）。
* **台灣數據**：$M \ge 4$ 的地震平均每週約有 **5 個**。
* **注意**：統計規律不一定帶有物理意義，「安靜期」與「活躍期」往往是馬後炮。

### 機率式地震危害度分析 (PSHA)
* **目的**：長期預報，用於防災部署、國家建設、耐震法規、地震保險費率制定。
* **圖表**：未來50年台灣孕震構造之發震機率圖（顯示各斷層發生大地震的機率）。

## 7. 特別案例：複合式災害
### 馬太鞍溪堰塞湖 (2024-2025)
這是一個典型的「地震 $\rightarrow$ 颱風 $\rightarrow$ 土石流/堰塞湖 $\rightarrow$ 潰壩」連鎖災害案例。
1.  **2024/4/3 大地震**：造成山體鬆動。
2.  **2025/7/21 薇帕颱風**：豪雨引發上游大規模邊坡崩塌，土石瞬間堵塞河道形成堰塞湖。
3.  **後續**：水位因降雨不斷升高。
4.  **2025/9/23 樺加沙颱風**：期間發生溢流和潰壩，造成下游洪災。

### 大規模破裂案例
* 由南至北破裂時間近 **500秒**。
* 破裂長度達 **1200公里**。
* 包含四次大型滑移事件。
* 破裂速度約 **2.9 - 3.0 km/s**。
