## 2. 何謂綠色科技？

* **宗旨：** 節能減碳、降低耗用能源、減少空氣污染。
* **範圍包括：**
    1.  **綠色能源** (如：太陽能)
    2.  **綠色食品** (如：有機、無農藥)
    3.  **綠色交通** (如：腳踏車、電動車)
    4.  **綠色建築**
    5.  **綠色照明** (如：發光二極體 LED)
    6.  **綠色材料** (如：水、樹葉、氧化鋅)
    7.  **綠色家電** (如：Motorola 可自動分解手機)

## 3. 相關背景：京都議定書

* 1997年由世界各國達成，於2005年2月16日正式生效，目標在遏止全球氣候暖化。
* **要點：**
    * **廢氣排放控制：** 控制六種溫室氣體 (二氧化碳、甲烷、一氧化二氮(笑氣)、氫氟碳化物、全氟化碳、六氟化硫)。
    * **目標：** 以1990年為基準，對35個工業國家訂定排放限制。
    * **交易：** 允許國家間進行溫室氣體排放量的「交易」。
    * **聯合作業：** 允許各國在其他工業國協助發展減少廢氣排放計畫。
    * **環境淨化發展機制：** 協助開發中國家發展降低廢氣排放計畫。

## 5. 綠色科技與光電元件應用

### (A) 綠色交通

* **定義：** 具有低污染、低噪音、省能源的綠色交通工具。
* **範例：** 電動車、腳踏車、搭公車、捷運。
* **電動車的缺點：** 電池壽命、電池價格、電池污染。

### (B) 太陽能 (節能、環保、再生能源)

#### 前言
* 隨工業發展，能源危機與環境污染問題浮現，利用太陽能是重要解決途徑。
* 近年太陽能發展迅速，但仍有**成本過高**、材料等因素，導致普及困難。
* **奈米技術** 被視為是達成更高效率、更低成本的研發新路。

#### 發電原理
1.  **結構：** 由 N 型半導體和 P 型半導體組成的太陽能半導體晶片。
2.  **受光：** 陽光照射晶片。
3.  **分離：** 晶片受光過程中，帶正電的「正孔」往 P 型區移動；帶負電的「電子」往 N 型區移動。
4.  **導電：** 電子從 N 區負電極流出負電；正孔從 P 區正電極流出正電。

#### 太陽能電池種類
* **市場主要產品：**
    * 矽晶圓太陽能電池
    * 非晶系矽太陽能電池
    * 銅銦鎵二硒 (CIGS) 太陽能電池
    * 碲化鎘 (CdTe) 薄膜太陽能電池
    * 矽薄膜太陽能電池
    * 染料敏化太陽能電池
    * 化合物 (GaAs) 太陽能電池
    * 奈米太陽能電池
    * 鈣鈦礦太陽能電池
* **矽晶太陽能電池比較：**
    * **單結晶矽：** 效率最高
    * **多結晶矽：** 較單晶便宜
    * **非結晶矽：** 價格最便宜

#### 種類詳述：染料敏化太陽能電池 (DSSC)
* **研發：** 1991 年由瑞士洛桑高等工業學校 (EPFL) 的 M. Grätzel 教授領導的小組所發展。
* **效率：** 當時的光電轉換效率達 7.1% (AM1.5)。
* **參考性能：**
    * $J_{sc}$ (短路電流密度): 12-13 mA/cm²
    * $V_{oc}$ (開路電壓): 0.68-0.73 V
    * $\eta$ (效率): 5.5-6.0%

#### 應用實例
* **太陽能車：**
    * **挑戰：** 太陽光強弱影響電流穩定性，需靠蓄電瓶與控制系統；此外，光電轉換效率低、成本高等因素。
    * **結論：** 以商品眼光來看，雖零污染但「沒有市場化的本錢」，始終停留在學術競賽。
* **已量產商品 (HIGHER WAY)：**
    * **太陽能 HID 35W + 8 LED 氙氣路燈**
        * **特點：** 簡易安裝、不需更換電池、應用面廣。
        * **規格：** HID 亮度 3200LM、蓄電池 DC12V、防護等級 IP66、太陽能發電 DC12V, 120W。
    * **太陽能地磚燈**
        * **特點：** 簡易安裝、不需更換電池、環保節能、耐撞擊。
        * **規格：** 太陽能發電 3V, 0.48A、高亮度 LED*4、尺寸 20x20x8cm。
