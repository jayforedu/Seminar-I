# 日期 : 2025/11/11
# 講者 : 欉振坤
# 題目 : 生成式人工智慧與異質平台整合應用
# 心得 :
本次演講深入分析了生成式人工智慧（GAI）的核心本質及其技術演進、驚人的成本、信任危機，直至其在產業中的實際應用定位。

演講首先闡明了GAI與傳統人工智慧的差異。 GAI的核心在於“生成”，使其能夠“創造原本不存在的數據”，而不是像過去那樣僅僅遵循固定的規則。 GAI的發展歷程也生動地展現了其能力的快速迭代——從早期基於文字輸入的模型發展到能夠處理多模態輸入（如圖像）並進行深度分析的高級模型——以及GAI的創造力和理解力如何迅速逼近人類。
然而,演講也毫不迴避 GAI 面臨的嚴峻挑戰。首先是驚人的訓練成本。當一個頂尖模型需要動用數千個高效能運算單元、進行長達數月的訓練時，這已明確宣告 GAI 的研發是資源高度密集的競賽。其次，是 GAI 特有的「信任」問題。演講中提到，GAI 傾向產生「幻覺」（虛擬內容），這正是 GAI 導入應用的最大絆腳石。這也呼應了建立「可信賴服務」的重要性，GAI 必須在可解釋性、穩健性、安全性等方面取得突破，才能真正被產業所接納。

最後，演講對 GAI 的應用提出了最務實的見解：正因為 GAI 成本高昂且存在信任問題，它目前最強大的定位並非全面取代，而是扮演「資源協調者」，串聯智慧化成果。以生產排程為例，GAI 不會取代既有的 APS 系統，而是賦能 APS。它讓生管人員能從 "As-Is"現況的繁瑣手動操作轉變為 "To-Be"未來——只需用自然語言"詢問 GAI"，GAI 就能主動介入、協調並整合所有資訊。

總之，這場演講讓我明白，生成式 AI 是一項革命性技術，但我們必須務實看待。它現階段的真正價值，是作為一個強大的「智慧介面」，串聯並優化現有的數位流程，使其更有效率。
